The source files of the `TBIA_Demo`Unreal Project can be downloaded
[here](https://static.twistedbytes.net/downloads/7a42299c-6047-e1e8-5b98-c16ab1868b89/TBIA%20Demo%20Project/).

The `TBIA_Demo` Unreal Project showcases how the **TwistedBytes Interaction System**
can be used. There are several scenarios grouped together in each room that
demonstrate different gameplay mechanics.
